Key Insights: Scenario 2 (Chatbot → MCP → API) is the primary integration pattern. Scenario 4 (Chatbot → API) is the simplest direct approach. Scenarios 1 & 3 show potential SDK integration if Bread provides a server-side SDK—currently unclear from docs. MCP Server provides standardized tool discovery and JSON-RPC interface for dynamic integration.
