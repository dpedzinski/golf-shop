🔄 Runtime Request Flow

"I want to shop for irons for experienced players"
1
👤 User Browser
User types message and clicks "Send" in chat widget
POST /sessions/web-UUID:runSession
{ "inputs": [{ "text": "I want irons for experienced players" }] }
↓
2
🤖 GECX Root Agent (Gemini LLM)
Analyzes: User wants irons + experienced skill level
Instruction says: Call search_products when category + skill provided
Decision: Prepare tool call
tool: "search_products"
args: { q: "irons", skill_level: "experienced", limit: 5 }
↓
3
🔀 CES Toolset Router
Looks up tool in golf-store-mcp toolset
Routes to MCP Server endpoint with JSON-RPC format
↓
4
🔌 MCP Server (Cloud Function)
Receives: JSON-RPC tool call
Translates to: REST API call
GET /products?q=irons&skill_level=experienced&limit=5
↓
5
⚡ Product API (Cloud Function)
Receives REST request and builds BigQuery SQL query
SELECT * FROM vw_product_listing_current
WHERE product_name LIKE '%irons%' AND
target_player_profile LIKE '%experienced%'
LIMIT 5
↓
6
📊 BigQuery
Executes query on golf_products dataset
Returns: 2 iron sets matching criteria (P027, P024)
↑ Results flow back up
7
🤖 Agent Processes Tool Result
Gemini receives: 2 products from catalog
Generates: Response text + rich content payload
Creates: product-carousel widget with product cards
↓
8
🌐 User Browser Renders
Text: "Here are iron sets that fit experienced players..."
Widget: <cx-product-carousel> with 2 interactive product cards
