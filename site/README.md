# Fairway Supply Site

Static storefront for the golf customer-service prototype.

The page reads these runtime environment variables for the GECX/Dialogflow Messenger embed:

- `GECX_PROJECT_ID`
- `GECX_LOCATION`
- `GECX_AGENT_ID`
- `GECX_LANGUAGE_CODE`
- `GECX_CHAT_TITLE`
- `GECX_ENABLE_WIDGET`
- `GECX_OAUTH_CLIENT_ID`

Run locally:

```bash
npm install
npm run dev
```
