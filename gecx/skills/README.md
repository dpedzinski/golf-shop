# GECX Skills

These files describe the customer-facing skills supported by the Golf Store
Assistant. They are metadata for the prototype and map each skill to the
BigQuery-backed MCP tools, product API endpoints, and storefront component
payloads used by the implementation.

Terraform reads the live agent behavior from
`gecx/agents/golf-store-assistant/instruction.txt`. The agent manifest at
`gecx/agents/golf-store-assistant/Golf_Store_Assistant.json` links to these
skill files for export, review, and future automation.
